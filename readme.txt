== Description ==

In partnership with Energy Storage Council, we (Global-Roam Pty Ltd) appreciate the opportunity
to serve you by making Australia's energy storage more broadly understood.

If you don't already have an API key you can get one [here](https://widgetadmin.global-roam.com/register/StorageCatalogue/66ec277d-e0f0-4fdb-822c-fcdb6ae9c466).
Alternately, you can move your mouse over the widget and click the **EMBED** button.

== Installation ==

If you have already received an email with a link to the embedding instructions, please follow them to find your API key.

1. Install the plugin as usual
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[battery-finder-widget token="your-api-token"]` on your page

If you don't already have an API key you can get one [here](https://widgetadmin.global-roam.com/register/StorageCatalogue/66ec277d-e0f0-4fdb-822c-fcdb6ae9c466).
Alternately, you can move your mouse over the widget and click the **EMBED** button.
